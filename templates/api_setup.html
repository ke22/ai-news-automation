🔑 API 金鑰設定

設定您的 API 金鑰以使用 AI 新聞自動化系統

📊 當前狀態

📝 設定步驟

方法一：使用設定腳本（推薦）

  1. 在終端機中執行：
    ./setup_env.sh
  2. 按照提示輸入您的 API 金鑰
  3. 重新啟動 Web 介面

方法二：手動設定

  1. 在終端機中執行以下命令：
    2. export GEMINI_API_KEY="your_actual_gemini_api_key_here"
    export NEWS_API_KEY="your_actual_news_api_key_here"
  2. 重新啟動 Web 介面

🔑 API 金鑰說明

🤖 Gemini AI API 金鑰（必需）

用於 AI 新聞評分、分類和內容生成。這是系統的核心功能。

獲取方式：

  1. 訪問 Google AI Studio
  2. 登入您的 Google 帳戶
  3. 點擊「Create API Key」
  4. 複製生成的 API 金鑰

📰 News API 金鑰（可選）

用於收集更多新聞來源。如果未設定，系統仍會使用 RSS 和 Hacker News。

獲取方式：

  1. 訪問 NewsAPI
  2. 註冊免費帳戶
  3. 在儀表板中複製 API 金鑰
